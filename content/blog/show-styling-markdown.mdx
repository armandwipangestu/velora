# Code Block

## Line Numbers and Line Highlighting

Draw attention to a particular line of code. This should be highlight the line number `4`.

```tsx title="example.tsx" {4} showLineNumbers
import { useFloating } from "@floating-ui/react";

function MyComponent() {

  return (
    <>
      <div ref={refs.setReference} />
      <div ref={refs.setFloating} style={floatingStyles} />

```

### Kitchen Sink Meta Strings

This should be highlight the meta strings, have some line numbers, and have a title and caption.

```ts showLineNumbers {2-4} title="isEven.ts" /console/ caption="Im a caption"
function isEven(number) {
  if (number === 1) {
    return false;
  } else if (number === 2) {
    return true;
  } else if (number === 3) {
    return false;
  } else if (number === 4) {
    return true;
  } else if (number === 5) {
    return false;
  } else if (number === 6) {
    return true;
  } else if (number === 7) {
    return false;
  } else if (number === 8) {
    return true;
  } else if (number === 9) {
    return false;
  } else if (number === 10) {
    return true;
  } else {
    return "Number is not between 1 and 10.";
  }
}
```

## Diff Highlighting

Show code changes with diff syntax highlighting. Lines starting with `+` are additions, `-` are deletions.

```diff
- const oldFunction = () => {
-   return "old value";
- };
+ const newFunction = () => {
+   return "new value";
+ };

  // Unchanged context line
- import { oldModule } from "./old";
+ import { newModule } from "./new";
```

## Group Highlighted Chars By Id

Place an id after # after the chars. This allows you to color chars differently based on their id.

```js showLineNumbers /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```

**Styling**: The chars `<span>` receives a `data-chars-id="<id>"` attribute to style via CSS.

## Inline Diff Highlighting

You can mark lines as additions or deletions in any language using `// [!code ++]` or `// [!code --]`.

```tsx showLineNumbers
function MyComponent() {
  return (
    <div>
      //[!code --:3]
      <h1>Hello World</h1>
      <p>Unchanged Line</p>

      //[!code ++]
      <h1>Hello Next.js</h1>

      <p>Highlighted Line</p>
      <p>Focused Line</p>
    </div>
  );
}
```

**Bold Text** and __Bold Text__

***Bold and Italic***

### Strikethrough

~~Strikethrough~~

### Inline Code

`Inline Code`

## Blockquotes

> This is a blockquote.
> It can span multiple lines.

> Blockquotes can be nested...
>> ...by using additional greater-than signs right next to each other...
>>> ...or with spaces between arrows.

Here an example blockquote with code block:

> **NOTE**: Here is a note
> 
> ```tsx
> const x = 10;
> ```
