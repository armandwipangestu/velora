# Callout

Velora make custom the callout component that inspired by GitHub callout markdown render. You can use it with this syntax:

```mdx
<Callout type="note">
    Here is a note callout
</Callout>
```

And it will be rendered as:

<Callout type="note">
    Here is a note callout
</Callout>
