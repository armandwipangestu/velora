## Type

There are 5 types of callout:

### Note

To use this you can passing the parameter `type` with value `note` to the props.

```mdx
<Callout type="note">
    Here is a note callout
</Callout>
```

It will be rendered as:

<Callout type="note">
    Here is a note callout
</Callout>
