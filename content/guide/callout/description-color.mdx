## Description Color

You can use the `descriptionColor` prop to set the description color of the callout.

```mdx
<Callout type="tip" descriptionColor="normal">
    Here is a tip callout with description color normal
</Callout>
```

It will be rendered as:

<Callout type="tip" descriptionColor="normal">
    Here is a tip callout with description color normal
</Callout>
